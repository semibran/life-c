# life-c
A minimalistic implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway's_Game_of_Life) in C. Uses SDL for graphics.

![screenshot](life.png)

## install
```sh
git clone git@github.com:semibran/life-c
cd life-c
mkdir dist
make
./dist/life
```
